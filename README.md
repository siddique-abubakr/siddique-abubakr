## What I've Owned at Litmustest

- **Plagiarism Detection (MOSS)** — celery task orchestration, status state machine, comparison UI, organization-level access control, language coverage (Php, Kotlin, Swift, Golang, React, Text etc), retry-with-cooldown UX.
- **Question Attachments** — upload, signed-URL retrieval, copy-on-update, ORA2 source-dir caching for the code-execution environment.
- **Question Authoring Stepper** — non-linear multi-step form, per-question stub codes, language restrictions, rubric-required gating.
- **Code Editor & Code Execution** — added PHP, Golang, React/JSX, Python-ML, Kotlin, and Swift; SQLite integration across multiple executors; CodeMirror syntax highlighting.
- **AI Rubric Evaluation** — OpenAI SDK + GPT-4o, per-question status, email-keyed identification.
- **Locust Load Testing Harness** — used to validate platform capacity before SOFTEC '25 and other bulk hiring events.
- **Accessibility (WCAG 2.1)** — skip navigation, focus indicators, ARIA labelling, keyboard support across the SPA, plus an upstream edX-platform focus-order fix.
- **Pylint CI Workflow** — score-gate, PR-comment integration scoped to changed files, qa-branch coverage.

540+ commits across five interconnected repositories.
