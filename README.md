## What I've Owned at Litmustest

- **Plagiarism Detection (MOSS)** — celery task orchestration, status state machine, comparison UI, organization-level access control, language coverage (Php, Kotlin, Swift, Golang, React, Text etc), retry-with-cooldown UX.
- **Question Attachments** — upload, signed-URL retrieval, copy-on-update, ORA2 source-dir caching for the code-execution environment.

- **Accessibility (WCAG 2.1)** — skip navigation, focus indicators, ARIA labelling, keyboard support across the SPA, plus an upstream edX-platform focus-order fix.
